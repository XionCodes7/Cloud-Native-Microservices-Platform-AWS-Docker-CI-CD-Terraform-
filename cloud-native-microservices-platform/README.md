# Cloud-Native Microservices Platform (AWS + Docker + CI/CD + Terraform)

Cloud-native microservices platform deployed on AWS using Docker containers, Terraform infrastructure-as-code, and CI/CD pipelines.

## Architecture

Client → API → Microservices → AWS → Logs

## Tech Stack
- Python (Flask)
- Docker
- AWS (ECS, EC2)
- Terraform
- GitHub Actions
